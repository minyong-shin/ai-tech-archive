# 영어 변수명 및 약자     

## 도메인    
- 금융, 마케팅 등 도메인을 기준으로 정함     
- 각각의 도메인에는 CTR, CPP 등 특정 지표를 2~4글자로 줄여쓰는 경우가 많기 때문에 해당 용어를 활용하여 변수 및 객체명을 명명하는 것을 추천     

- 상품 : prod     
- 부서/부처 : dept     
- 매출 : rev     
- 송금/변환 : trans     
- 인구 : pop     
- 교육 : edu     

## 단위    
- 개수 : cnt     
- 단위 : unit     
- 요일 : weekday     
- 주말 : weekend     
- 시간 : hour, hr     
- 분 : min      
- 초 : sec     

## 연산     
- 평균, 분산 등 통계량을 표현할 때는 접미사를 주로 씀     
- 합계 : sum     
- 최대 : max    
- 최소 : min    
- 평균 : mean, avg     
- 분산 : var     
- 표준편차 : sd, std     
- 왜도 : skew     
- 첨도 : kurt     
- 분포 : dist     
- 요약 : agg    
- 빈도 : freq     
- 비율 : prop, ratio     
- 확률 : prob    
- 모집단 : pop    
- 개수/수량 : cnt, amt     

## 객체 속성에 따른 명명     
- 1차원 vector : vec     
- matrix : mat    
- dataframe : df     
- list : list, ls    

## 표기 순서    
- 객체 속성 -> 데이터 도메인/출처 순이 일방적인 방식     

## 객체 필터링에 따른 명명    
- 특정 객체에서 필터링하여 일부를 뽑아낸 객체는 접미사로 해당 내용을 명시     
- 일반적인 필터링 결과 : sub    
~~~
df_stat_sub = df_stat[2:6, ]     
~~~
